# SME Grant Management System - Uva Province

A comprehensive digital platform for managing Small and Medium Enterprise (SME) grants in the Uva Province, Sri Lanka.

## Key Features
- **Dynamic Scoring Engine**: Automated eligibility assessment based on administrator-configured weights.
- **Multi-level Approval Workflow**: Seamless transitions between Development Officers, Divisional Secretaries, and Directors.
- **Mobile-First Design**: Optimized for Development Officers to enter data via smartphones in the field.
- **Administrative Governance**: Complete control over staff roles, regional sectors, and grant policies.
- **Secure Authentication**: Role-based access control with account deactivation capabilities.

## Tech Stack
- **Frontend**: React, Framer Motion, Lucide React
- **Backend**: Firebase Firestore, Firebase Auth
- **Styling**: Vanilla CSS with modern glassmorphism aesthetics

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Configure your Firebase credentials in `src/firebase.js`
4. Start the dev server: `npm run dev`
